# ourproject
Product Management System (CRUD)

This is a simple Product Management System built to manage products with basic CRUD operations.
The project includes the following main features:

Add new products

Update existing products

Delete products

Search for products (by name)

Display a full product list

Data is stored locally on the browser using LocalStorage

Features
➤ Add Product

You can add a new product by entering its name, price, category, and description. The form validates inputs before saving.

➤ Update Product

Any product in the list can be edited. When you click update, the data is loaded back into the form where you can modify it and save changes.

➤ Delete Product

You can delete a single product or clear all products at once.

➤ Search

The search bar lets you search for products in real time to quickly filter the list.

➤ Product List

All added products are displayed in a clean table layout, showing all details with options to update or delete each product.
